# Sistema Clínico

Sistema web para gestão de agendamentos de consultas e controle de pagamentos em clínicas médicas.

## 👥 Integrantes
- Geovanna Alves Rodrigues
- Luan Matos
- Lucas Kauan
- José Carlos

## 🎯 Objetivo
Otimizar processos administrativos de clínicas, reduzindo erros de agendamento e facilitando o controle financeiro.

## ⚙️ Principais Funcionalidades
- Cadastro de pacientes e médicos
- Agendamento e cancelamento de consultas
- Registro de atendimentos
- Controle de pagamentos
- Histórico do paciente
- Relatórios gerenciais
- Envio de notificações por e-mail

## 🛠️ Tecnologias
- Python + Django
- Tailwind CSS
- MySQL
- JWT para autenticação

## 🗄️ Banco de Dados
Estrutura principal:
- usuarios – controle de acesso (ADMIN/FUNCIONARIO)
- clientes – cadastro de pacientes
- vendas – registro de pagamentos
- produtos / intens_venda – para expansão futura
